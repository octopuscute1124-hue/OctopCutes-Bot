# :octopus: OctopCutes 防傳銷機器人

一個專為 Discord 設計的防傳銷/防騷擾機器人，具備全域黑名單與自動封鎖功能。

---

## :sparkles: 功能特色

- :shield: **頻道保護**：指定頻道一發言即 Ban
- :clipboard: **全域黑名單**：JSON 儲存，跨伺服器同步封鎖
- :octagonal_sign: **止損機制**：1秒內2頻道 @everyone/@here 自動 Ban
- :video_game: **直覺操作**：`!章魚` 叫出控制面板，下拉選單輕鬆設定
- :bar_chart: **黑名單查詢**：一鍵查看所有被封鎖帳號
- :arrows_counterclockwise: **定期掃描**：每30分鐘自動掃描所有伺服器
- :floppy_disk: **設定持久化**：重啟機器人保護設定不消失

---

## :rocket: 快速開始

### 安裝

```CMD
git clone https://github.com/octopuscute1124-hue/OctopCutes-Bot.git
cd OctopCutes-Bot
npm install
```

### 設定

1. 複製 `.env.example` 為 `.env`
2. 填入你的 Discord Token

```env
DISCORD_TOKEN=你的機器人Token
```

### 啟動

```bash
node bot.js
```

## [V0.2.0] - 2026-07-22

### 🎉 重大更新：完全重構為純防護系統

本次更新移除頻道保護功能，將機器人轉型為**全方位自動安全防護系統**，並對程式碼進行大規模重構與優化。

### ✨ 新增功能

#### 🛡️ 安全防護 (22 項)
- **止損機制**：偵測短時間內在 2 個以上頻道 @everyone/@here，自動 Ban
- **@mention 速度檢測**：偵測快速大量 @ 人行為，自動 Ban
- **惡意腳本檢測**：偵測短時間大量發送訊息，自動 Ban
- **SelfBot 檢測**：偵測快速回應行為，自動 Ban
- **洪水攻擊防護**：偵測極短時間大量發送，自動 Ban
- **爬蟲行為檢測**：偵測大量請求，自動 Timeout
- **語音頻道濫用**：偵測快速切換語音頻道，自動 Ban
- **非人類權限變更**：偵測快速變更頻道權限，自動警報 + 復原
- **Webhook 濫用監控**：偵測 Webhook 大量發送，自動刪除
- **頻道建立/刪除濫用**：偵測大量建立/刪除頻道，自動警報
- **撞庫攻擊防護**：偵測短時間多次加入嘗試，自動 Ban
- **洪水加入防護**：偵測大量成員加入，自動警報
- **邀請連結濫用**：偵測大量建立邀請，自動 Ban
- **可疑帳號檢測**：偵測新創帳號 / 無頭像帳號，警報 / 自動 Ban
- **暴力破解防護**：偵測大量失敗嘗試，暫時封鎖
- **惡意檔案檢測**：阻擋 .exe, .bat, .js 等惡意檔案
- **XSS/HTML 注入**：阻擋 script, eval, javascript: 等惡意代碼
- **惡意 Rich Presence**：偵測包含可疑程式碼的活動狀態，自動 Ban
- **角色權限鎖定**：防止非管理員角色被賦予管理員權限
- **Rate Limiting**：防止短時間大量 API 請求
- **行為追蹤**：偵測異常操作模式
- **自動降級**：系統壓力過大時自動限縮功能

#### 🔔 警報系統 (9 項)
- 可疑帳號加入警報
- 洪水加入警報
- Webhook 濫用警報
- 權限異常變更警報
- 機器人被 Ban 警報
- 頻道建立/刪除警報
- 角色權限濫用警報
- 邀請連結濫用警報
- 惡意 Rich Presence 警報

#### 🎮 控制面板
- `!章魚` 主控制面板
- 安全功能獨立開關 (22 項)
- 警報功能獨立開關 (9 項)
- 黑名單管理 (僅開發者)
- 白名單管理 (僅開發者)
- 自動回應管理 (僅開發者)
- 設定匯出功能 (僅開發者)
- 警報頻道設定

#### 📊 資料儲存
- `blacklist.json`：全域黑名單
- `config.json`：所有設定 (白名單、安全開關、警報開關、警報頻道、自動回應)
- `logs.json`：操作日誌、Ban 紀錄、管理員操作紀錄
